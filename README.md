# 💰 Expense Tracker (Python CLI Project)

🚀 This is a simple **Command-Line Expense Tracker** built using Python as part of my Internship Task 2.

---

## 📌 Project Overview

This project helps users to:
- Add daily expenses
- View saved expenses
- Calculate total spending

All data is stored in a **CSV file**, making it simple and efficient.

- Python Basics
- Functions
- File Handling (Read & Write)
- CSV Format
- Loops and Conditions
